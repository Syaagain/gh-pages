Utility functions for using OpenAI gymnasium

seprlhf.utils.gym.make_gymnasium_env(env_id: str, num_envs: int, capture_video: bool, run_name: str, gamma: float) → List[SyncVectorEnv][source]

Creates an OpenAI gymnasium environment based on a env ID.

Parameters:

  • env_id (str) – ID for gymnasium environment -> e.g. “CartPole-v1”

  • num_envs (int) – Number of parallel environments.

  • capture_video (bool) – Whether or not a video should be captured while training.

  • run_name (str) – Specify a (unique) name for every run.

  • gamma (float) – Discount factor for reward normalization.

Returns:

Returns a list of vectorized gym environments.

Return type:

List[gym.vector.SyncVectorEnv]

seprlhf.utils.gym.render_actions(env_name: str, actions: Tensor, video_dir: str, seg_nr: int, fps: int = 30)[source]

Render actions as frames and save them into a video file.

Parameters:

  • env_name (str) – Name of the gym environment.

  • actions (torch.Tensor) – action tensor.

  • video_dir (str) – Path to the output file.

  • seg_nr (int) – 1 or 2, to compare segments/videos afterwards

  • fps (int, optional) – Frames per second for the video, default: 30
